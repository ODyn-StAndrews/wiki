Download location: [input4MIPs](https://esgf-node.llnl.gov/search/input4mips/)  

**Download procedure**
* At [input4MIPs](https://esgf-node.llnl.gov/search/input4mips/), select `"Target MIP" = "OMIP"`, `"Institution ID" = "MRI"`, and `"Source Version" = "1.5.0"` among tabs on the left side.
* Login with ESGF credentials (or create new account) and add all desired variables to Data Cart
* In Data Cart, select Globus Download.
* Select Web Browser option, which will navigate to the Globus webpage (login with appropriate institution ID).
* Select appropriate Globus collection to transfer data to (e.g. Princeton TIGRESS) and approve transfer.
