# Xarray  

`xarray` is a Python package developped for analysing multi-dimensional gridded datasets, with several advantages over numpy such as labelling dimensions and attributes (e.g., "units", "long_name", etc.)

A good review of commonly used features of xarray (with examples and exercises) can be found on this Project Pythia [page](https://foundations.projectpythia.org/core/xarray.html).
